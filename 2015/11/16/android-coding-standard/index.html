Android编码规范（非官方），持续更新中

  • Android

这里只是个人习惯的归档，并不是官方规范。包括两大部分：Java代码规范和资源文件命名规范。

Java Part

类成员变量

统一以’m’作为前缀来命名类成员变量。当然有唯一例外，那就是POJO或者JavaBean。

成员变量初始化

若成员变量以’m’前缀，则成员变量初始化不带’this’关键字；若类作为POJO或者JavaBean形式，则仍然以’this’关键字的形式初始化。
